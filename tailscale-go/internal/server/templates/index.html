使用說明

Tailscale 是一種零配置 VPN，可讓您在不同設備和網絡之間建立安全連接。使用 Tailscale，您可以從任何地方安全地訪問您的 Home Assistant。

主要功能:

  • 從任何地方安全地訪問您的 Home Assistant
  • 無需端口轉發或 DynamicDNS
  • 連接到您家中的所有智能家居設備
  • 在公共 Wi-Fi 上安全地瀏覽互聯網

更多信息請參閱 GitHub 存儲庫。
